# 💰 BNB Price Predictor

A Machine Learning web application that predicts the **closing price of Binance Coin (BNB)** using market indicators:

- Open Price
- High Price
- Low Price
- Volume

Built with **Linear Regression**, deployed using **Flask + Vercel**.

## 📌 Project Overview

This project demonstrates how Machine Learning models can be integrated into a real-world web application.

The model is trained on historical BNB market data and predicts the **Close Price** based on user-provided input features.

---

## 🧠 Machine Learning Details

- Algorithm: Linear Regression
- Feature Scaling: MinMaxScaler
- Features Used:
  - Open
  - High
  - Low
  - Volume
- Target Variable:
  - Close Price

Evaluation Metrics:
- MAE (Mean Absolute Error)
- MSE (Mean Squared Error)
- RMSE
- R² Score

> Note: This project is for educational purposes. In real-world financial forecasting, time-series models like LSTM or ARIMA are more appropriate.
